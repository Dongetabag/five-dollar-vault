## BLOG POST PROMPTS

### PROMPT 1: Blog Post Outline Generator

```
Create a comprehensive blog post outline on this topic:

**Topic:** [YOUR TOPIC]
**Target Audience:** [WHO'S READING]
**Primary Keyword:** [SEO KEYWORD]
**Word Count Target:** [1000/1500/2000]
**Goal:** [EDUCATE/PERSUADE/ENTERTAIN]
**CTA:** [WHAT SHOULD READERS DO AFTER]

Generate:
1. **5 Headline Options** (include numbers, power words, curiosity)
2. **Meta Description** (155 characters, keyword included)
3. **Hook/Introduction** (3 opening approaches)
4. **H2 Subheadings** (6-8 sections with descriptions)
5. **Key Points** (bullet points under each H2)
6. **Internal Link Opportunities** (where to link to other content)
7. **External Link Suggestions** (authoritative sources to cite)
8. **Conclusion Approach** (summarize + CTA)
9. **Featured Image Concept** (what visual to use)
```

### PROMPT 2: Blog Post Writer (Full Draft)

```
Write a complete blog post based on this outline:

**Headline:** [YOUR CHOSEN HEADLINE]
**Outline/Structure:** [PASTE YOUR OUTLINE OR KEY POINTS]
**Tone:** [PROFESSIONAL/CASUAL/AUTHORITATIVE/FRIENDLY]
**Target Audience:** [WHO'S READING]
**Include:** [ANY SPECIFIC EXAMPLES, DATA, OR STORIES]

Requirements:
- Start with a hook that doesn't begin with "In today's world" or similar clichÃ©s
- Use short paragraphs (2-3 sentences max)
- Include bullet points and numbered lists for scannability
- Add a "Key Takeaway" callout box after major sections
- Use conversational language (contractions okay)
- Include one relevant analogy or metaphor
- End with a strong CTA
- Word count: [TARGET]

Do NOT:
- Use the phrase "dive into" or "deep dive"
- Start paragraphs with "So" or "Now"
- Use excessive exclamation points
- Be generic - be specific and opinionated
```

### PROMPT 3: Listicle Generator

```
Create a listicle blog post:

**Topic:** [X] [THINGS/TIPS/TOOLS/MISTAKES/WAYS] to [OUTCOME]
**Number of Items:** [5/7/10/15]
**Target Audience:** [WHO'S READING]
**Expertise Level:** [BEGINNER/INTERMEDIATE/ADVANCED]

For each list item provide:
- **Catchy Sub-headline** (benefit-focused)

- **Pro Tip** (advanced insight for that item)
- **Common Mistake** (what to avoid)

Also include:
- Compelling introduction (why this matters NOW)
- Summary table at the end
- "Start Here" recommendation for beginners
```

## SOCIAL MEDIA PROMPTS

### PROMPT 4: LinkedIn Post Generator

```
Write a LinkedIn post about:

**Topic/Story:** [WHAT YOU WANT TO SHARE]
**Hook Type:** [QUESTION/BOLD STATEMENT/STORY/STATISTIC]
**Goal:** [ENGAGEMENT/AUTHORITY/LEADS/TRAFFIC]
**CTA:** [COMMENT/SHARE/LINK/DM]

Requirements:
- First line must stop the scroll (under 150 characters)
- Use line breaks for readability (one sentence per line)
- Include a personal story or specific example
- Add 3-5 relevant hashtags at the end
- End with a question or clear CTA
- Length: 150-300 words (optimal for LinkedIn)

Provide 3 variations:
1. **Story-Based** (personal narrative)
2. **Value-Based** (tips/lessons)
3. **Contrarian** (challenge conventional thinking)
```

### PROMPT 6: Instagram Carousel Script

## VIDEO SCRIPT PROMPTS

### PROMPT 8: YouTube Video Script (Long-Form)

```
Write a YouTube video script:

**Title:** [VIDEO TITLE]
**Length:** [8-10 MIN / 15-20 MIN / 30+ MIN]
**Topic:** [WHAT THE VIDEO COVERS]
**Target Audience:** [WHO'S WATCHING]
**Video Style:** [TALKING HEAD/TUTORIAL/DOCUMENTARY]

Structure:
1. **Cold Open (0:00-0:30):** Hook that previews the value
2. **Intro (0:30-1:00):** Who you are, what they'll learn
3. **Why This Matters (1:00-2:00):** Stakes and context
4. **Main Content (2:00-X:00):** Core teaching (break into chapters)
5. **Recap (X:00-X:30):** Summary of key points
6. **CTA (X:30-end):** Subscribe, comment, watch next

For each section include:
- Script (word-for-word what to say)
- B-roll suggestions
- On-screen graphics/text
- Timestamps for chapters

Also provide:
- Video description (SEO optimized)
- Tags (15-20)
- Thumbnail concept
- Pinned comment draft
```

### PROMPT 9: Webinar/Presentation Script

```
Create a webinar script:

**Title:** [WEBINAR TITLE]
**Duration:** [30/45/60/90 MINUTES]
**Topic:** [WHAT YOU'RE TEACHING]
**Audience:** [WHO'S ATTENDING]
**Goal:** [TEACH/SELL/LAUNCH]
**Offer:** [IF SELLING, WHAT'S THE OFFER]

Structure:
1. **Pre-Show (5 min):** Housekeeping, tech check
2. **Opening (5 min):** Hook, agenda, credibility
3. **Content Block 1 (15 min):** [TOPIC]
4. **Content Block 2 (15 min):** [TOPIC]
5. **Content Block 3 (15 min):** [TOPIC]
6. **Transition to Offer (5 min):** Bridge from content to pitch
7. **Offer Presentation (10 min):** Features, benefits, price
8. **Q&A (15 min):** Handle objections
9. **Close (5 min):** Final CTA, urgency

For each section:
- Talking points (not word-for-word)
- Slide suggestions
- Engagement prompts (polls, questions)
- Transition phrases
```

## EMAIL PROMPTS

### PROMPT 10: Email Newsletter Template

```
Write a weekly newsletter email:

**Newsletter Name:** [YOUR NEWSLETTER]
**This Week's Theme:** [MAIN TOPIC]
**Main Content:** [KEY STORY/LESSON]
**Resources to Share:** [LINKS/TOOLS/ARTICLES]
**Personal Update:** [OPTIONAL: WHAT YOU'VE BEEN UP TO]
**CTA:** [REPLY/CLICK/BUY/SHARE]

Structure:
1. **Subject Line:** (3 options - curiosity, benefit, news)
2. **Preview Text:** (40-90 characters)
3. **Opening Hook:** (personal, timely, relatable)
4. **Main Content:** (the meat - 200-400 words)
5. **Resource Section:** (bulleted links with context)
6. **Personal Note:** (humanizing touch)
7. **CTA:** (clear ask)
8. **P.S.:** (secondary offer or reminder)

Tone: Like writing to a smart friend
Length: 500-800 words total
```

### PROMPT 11: Email Sequence (5-Part Nurture)

```
Create a 5-email nurture sequence:

**Lead Magnet/Entry Point:** [WHAT THEY SIGNED UP FOR]
**End Goal:** [BOOK CALL/BUY PRODUCT/JOIN PROGRAM]
**Audience:** [WHO THEY ARE]
**Timeframe:** [DAILY/EVERY 2 DAYS/WEEKLY]

Email 1 - Welcome + Delivery:
- Deliver the thing they signed up for
- Set expectations for future emails
- Quick win they can implement now

Email 2 - Your Story:
- Why you do what you do
- Relatable struggle
- Credibility without bragging

Email 3 - Value + Education:
- Teach something actionable
- Position as expert
- Light mention of your offer

Email 4 - Case Study/Social Proof:
- Customer success story
- Specific results with numbers
- "If they can, you can" message

Email 5 - The Offer:
- Direct pitch
- Clear benefits
- Urgency/scarcity (if real)
- FAQ objection handling
- Strong CTA

Provide subject lines, preview text, and full body for each.
```

### PROMPT 12: Sales Email Sequence (Cold Outreach)

```
Create a 4-email cold outreach sequence:

**My Company:** [WHAT YOU DO]
**Target:** [WHO YOU'RE EMAILING]
**Their Pain Point:** [PROBLEM YOU SOLVE]
**Goal:** [BOOK MEETING/GET REPLY]

Email 1 - The Opener:
- Pattern interrupt subject line
- Lead with their problem
- Brief intro to solution
- Soft CTA (reply)

Email 2 - The Value Add (3 days later):
- Reference Email 1
- Share relevant insight/resource
- No hard pitch
- Question to engage

Email 3 - The Case Study (5 days later):
- Quick success story
- Specific results
- "Similar situation to yours"
- CTA: 15-minute call

Email 4 - The Breakup (7 days later):
- Acknowledge silence is answer
- One last value add
- Open door for future
- Light/slightly humorous tone

Each email under 100 words. Subject lines under 40 characters.
```

## Pro Tips for Using These Prompts

1. **Stack Prompts:** Use the outline prompt, then the writer prompt
2. **Be Specific:** The more detail you give, the better output you get
3. **Iterate:** Ask for variations, revisions, different angles
4. **Save Winners:** Keep a swipe file of outputs that work
