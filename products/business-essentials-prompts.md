## PROMPT 1: Sales Email Generator

```
You are an expert B2B sales copywriter. Write a cold outreach email with these parameters:

**My Company:** [YOUR COMPANY]
**What We Sell:** [YOUR PRODUCT/SERVICE]
**Target Recipient:** [THEIR ROLE/COMPANY TYPE]
**Their Pain Point:** [PROBLEM YOU SOLVE]
**Desired Action:** [BOOK CALL/REPLY/VISIT SITE]

Requirements:
- Subject line under 50 characters (include a number or question)
- Opening line must NOT start with "I" or mention my company
- Reference a specific challenge they likely face
- Keep under 100 words
- End with a single, clear CTA
- P.S. line with social proof or urgency

Write 3 variations: formal, conversational, and bold.
```

## PROMPT 2: Competitor Analysis Deep Dive

```
Act as a competitive intelligence analyst. Analyze this competitor:

**Competitor:** [COMPANY NAME]
**Their Website:** [URL]
**My Company:** [YOUR COMPANY]
**Our Differentiator:** [WHAT MAKES YOU DIFFERENT]

Provide:
1. **Positioning Analysis:** How they position themselves (3 sentences)
2. **Pricing Strategy:** What can you infer about their pricing model?
3. **Target Customer:** Who are they selling to?
4. **Strengths:** 3 things they do well
5. **Weaknesses:** 3 gaps or vulnerabilities
6. **Messaging Themes:** Key phrases/words they repeat
7. **Attack Angles:** 3 ways I can position against them
8. **Content Strategy:** What content are they creating?

Format as a strategic briefing document.
```

## PROMPT 3: Customer Persona Builder

```
Create a detailed customer persona based on these inputs:

**Industry:** [TARGET INDUSTRY]
**Company Size:** [EMPLOYEE COUNT/REVENUE]
**Role I Sell To:** [JOB TITLE]
**Problem I Solve:** [CORE PAIN POINT]
**Price Point:** [YOUR PRICING]

Generate a complete persona including:
- **Name & Background** (fictional but realistic)
- **Demographics** (age, location, education)
- **Day in the Life** (typical workday challenges)
- **Goals & Motivations** (professional and personal)
- **Frustrations & Pain Points** (be specific)
- **Objections to Buying** (why they might say no)
- **Information Sources** (where they learn, who they trust)
- **Decision-Making Process** (who else is involved?)
- **Trigger Events** (what makes them start looking for solutions?)
- **Preferred Communication** (email, phone, LinkedIn?)

Make this persona feel like a real person I can picture.
```

## PROMPT 4: Meeting Agenda Creator

```
Create a professional meeting agenda:

**Meeting Type:** [SALES CALL/KICKOFF/STRATEGY/REVIEW]
**Duration:** [30/60/90 MINUTES]
**Attendees:** [WHO'S ATTENDING]
**Meeting Goal:** [WHAT SUCCESS LOOKS LIKE]
**Context:** [ANY BACKGROUND INFO]

Generate:
1. **Pre-Meeting Prep** (what attendees should do before)
2. **Timed Agenda** (specific time blocks adding up to duration)
3. **Discussion Questions** (2-3 per agenda item)
4. **Decision Points** (what needs to be decided)
5. **Next Steps Template** (to fill in during meeting)
6. **Follow-Up Email Draft** (to send after)

Format professionally with clear headers and time stamps.
```

## PROMPT 5: Proposal Executive Summary Writer

```
Write a compelling executive summary for a business proposal:

**Client:** [CLIENT COMPANY]
**Their Challenge:** [PROBLEM THEY HAVE]
**Our Solution:** [WHAT WE'RE PROPOSING]
**Investment:** [PRICE/PRICING MODEL]
**Timeline:** [IMPLEMENTATION TIMEFRAME]
**Key Stakeholder:** [WHO'S READING THIS]

Write a 250-word executive summary that:
- Opens with their problem (not our solution)
- Quantifies the cost of inaction
- Presents our solution as the logical answer
- Includes 2-3 expected outcomes with metrics
- Creates urgency without being pushy
- Ends with a clear next step

Tone: Confident, consultative, not salesy.
```

## PROMPT 6: FAQ Generator

```
Generate a comprehensive FAQ section for:

**Product/Service:** [WHAT YOU SELL]
**Target Customer:** [WHO BUYS IT]
**Price Point:** [PRICING]
**Common Objections:** [LIST ANY YOU KNOW]
**Competitors:** [MAIN ALTERNATIVES]

Create 15 FAQs covering:
- **Pre-Purchase** (5 questions about fit, pricing, comparison)
- **Implementation** (3 questions about getting started)
- **Usage** (4 questions about using the product)
- **Support** (3 questions about help and guarantees)

For each FAQ:
- Write the question as a customer would ask it
- Provide a clear, confident answer (50-100 words)
- Include a subtle CTA where appropriate

Format for easy website implementation.
```

## PROMPT 7: Performance Review Self-Assessment

```
Help me write my self-assessment for a performance review:

**My Role:** [JOB TITLE]
**Review Period:** [TIMEFRAME]
**Key Accomplishments:** [LIST 3-5 WINS]
**Challenges Faced:** [WHAT WAS HARD]
**Skills Developed:** [WHAT YOU LEARNED]
**Goals for Next Period:** [WHAT YOU WANT TO ACHIEVE]

Write a professional self-assessment that:
- Leads with quantified achievements (numbers!)
- Acknowledges challenges without making excuses
- Shows growth mindset and self-awareness
- Aligns personal goals with company objectives
- Positions me for [PROMOTION/RAISE/NEW PROJECT]

Length: 400-500 words
Tone: Confident but not arrogant, specific not vague
```

## PROMPT 8: Process Documentation Template

```
Create clear documentation for this business process:

**Process Name:** [WHAT IT'S CALLED]
**Purpose:** [WHY THIS PROCESS EXISTS]
**Who Performs It:** [ROLE/TEAM]
**Frequency:** [HOW OFTEN]
**Tools Used:** [SOFTWARE/SYSTEMS]
**Current Pain Points:** [WHAT'S BROKEN]

Generate:
1. **Process Overview** (2-3 sentences)
2. **Prerequisites** (what's needed before starting)
3. **Step-by-Step Instructions** (numbered, detailed)
4. **Decision Points** (if X then Y scenarios)
5. **Common Mistakes** (and how to avoid them)
6. **Quality Checklist** (verification steps)
7. **Escalation Path** (when to ask for help)
8. **Metrics** (how to measure success)

Format for a Standard Operating Procedure (SOP) document.
```

## PROMPT 9: Quarterly Business Review Deck Outline

```
Create a QBR presentation structure:

**Company/Department:** [YOUR AREA]
**Quarter:** [Q1/Q2/Q3/Q4 YEAR]
**Audience:** [WHO'S WATCHING]
**Key Metrics:** [YOUR KPIS]
**Highlights:** [WINS THIS QUARTER]
**Challenges:** [WHAT DIDN'T GO WELL]
**Next Quarter Goals:** [WHAT YOU'RE TARGETING]

Generate a 12-slide deck outline:
- **Slide 1:** Title & Quarter Overview
- **Slides 2-4:** Key Metrics (what to show, how to visualize)
- **Slides 5-6:** Wins & Highlights (stories to tell)
- **Slide 7:** Challenges & Learnings (honest assessment)
- **Slides 8-9:** Market/Competitive Updates
- **Slides 10-11:** Next Quarter Plan (goals, initiatives)
- **Slide 12:** Ask & Discussion

For each slide, include:
- Headline recommendation
- Key data points to include
- Visual/chart suggestions
- Talk track notes (what to say)
```

## PROMPT 10: Crisis Communication Draft

```
Help me draft a communication for a sensitive business situation:

**Situation:** [WHAT HAPPENED]
**Impact:** [WHO'S AFFECTED AND HOW]
**Audience:** [CUSTOMERS/EMPLOYEES/PARTNERS/PUBLIC]
**What We're Doing:** [ACTIONS BEING TAKEN]
**Timeline:** [WHEN THINGS WILL BE RESOLVED]
**Tone Required:** [APOLOGETIC/INFORMATIVE/REASSURING]

Draft a communication that:
- Acknowledges the issue directly (no corporate speak)
- Takes appropriate responsibility
- Explains what happened (without over-explaining)
- Details specific actions being taken
- Provides a timeline or next update commitment
- Offers a path for questions/concerns
- Maintains trust and credibility

Provide:
1. **Short Version** (for social media/quick update - 280 chars)
2. **Medium Version** (for email - 150 words)
3. **Full Version** (for website/press - 300 words)
```
